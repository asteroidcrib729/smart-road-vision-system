### This File (Project Description.md) Contains The Directory Structure Of The Smart Road Vision System Project.

1. The Folder Named "Every Iteration of 'main.py' File" Contains Each Version of The Code Contained In The 'main.py' File That Has Been Executed Starting From The Initial Baseline Version Up Till The Latest Version.
2. The Folder Named "SmartRoadVisionSystem" Is The Main Project Directory Where The Project's Current (Latest) Progress Can Be Found.
3. The Folder Named "SmartRoadVisionSystem (Baseline Version)" Contains The Initial Version Of The Project That Was Devised During The Construction Of Project's Development Plan.
4. The Zip File Named "Output Snapshots Collected During 07th Iteration of 'main.py' File.zip" Contains The Vehicle Snapshots That Were Autonomously Taken During Code (main.py) Execution.
5. The "requirements.txt" File Contains All The Packages That Were Installed During The Project's Development In Visual Studio Code For Installation Purposes In The Newly Created Virtual Environment.
